Playing Homework:Hotel Reservation

Overview
This is a simple one-page WEB APP based on angularjs to solve Hotel Reservation problem.

Prerequisites
These components are essential:
Git
Node.js
Grunt
Bower
Yeoman

How to use 

step1
cd dir to root directory,use npm -install to update the necessary components.

step2
use grunt test to run the test.

step3
use grunt server to run the app in local.

step4
chose your costomer type,and add date that your want to book,final click Find button to solve the problem.

Summary
By using automated build tools and package management tools,Construction projects can be simple and efficient.Main code is in \app\scripts\controllers\main.js.and test code is in \test\spec\controllers\main.js.
